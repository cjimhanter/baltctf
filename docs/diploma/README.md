# Черновик ВКР по подглавам

Этот каталог содержит Markdown-черновик выпускной квалификационной работы по проекту `baltctf`. Структура взята по образцу файла `docs/Диплом Семенов.docx`: титульная часть, содержание, три основные главы, заключение, список источников и приложения. Текст разбит на отдельные файлы по подглавам, чтобы их можно было дописывать независимо и затем собрать в один документ.

## Тема

**Разработка платформы для проведения соревнований по информационной безопасности в формате Attack/Defense CTF**

## Что нужно уточнить перед финальной версткой

- ФИО студента, научного руководителя, должности и регалии.
- Точное название образовательной программы и кафедры/школы.
